## Usage

Browse by country folder to find the team logo you need. Each folder contains SVG files named after the club or league.

### Example

To find the Arsenal logo:

```
england/Arsenal.svg
```

## Direct Usage

You can use logos directly from GitHub using the raw URL format:

```
https://raw.githubusercontent.com/JoseArroyave/football-logos/refs/heads/main/logos/{country}/{logo_filename}
```

### Example

```html
<img src="https://raw.githubusercontent.com/JoseArroyave/football-logos/refs/heads/main/logos/england/Leicester.svg" alt="Leicester Logo" />
```

Output:

![Leicester](https://raw.githubusercontent.com/JoseArroyave/football-logos/refs/heads/main/logos/england/Leicester.svg)
